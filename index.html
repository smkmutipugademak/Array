1ï¸âƒ£ Pengertian List

                    

                        List adalah tipe data di Python yang digunakan untuk menyimpan banyak nilai
                        dalam satu variabel.
                        List bisa berisi data dengan tipe yang berbeda seperti string, angka, atau
                        bahkan list lain.
                    

                    
contoh = ["apel", "jeruk", "mangga", 10, True]
print(contoh)

                    
Output:

                    
['apel', 'jeruk', 'mangga', 10, True]

                    

                        List bersifat mutable, artinya kita bisa menambah, menghapus, atau mengubah
                        isinya setelah dibuat.
                    

                    
ğŸ’¡ Analogi: Anggap List seperti â€œkotak penyimpananâ€ yang bisa
                        diisi berbagai benda dan bisa diubah isinya kapan saja.

3ï¸âƒ£ Mengakses Elemen dalam List

                    
Gunakan indeks untuk mengambil data tertentu dari list. Ingat, indeks pertama dimulai dari
                        0.
                    

                    
buah = ["apel", "pisang", "jeruk", "mangga"]
print(buah[0])   # apel
print(buah[2])   # jeruk
print(buah[-1])  # mangga (indeks dari belakang)

                    
âš ï¸ Jika mengakses indeks yang tidak ada, akan muncul error

4ï¸âƒ£ Menambah Elemen ke dalam List

                    
Python menyediakan beberapa cara untuk menambahkan elemen:

                    
buah = ["apel", "pisang"]

buah.append("jeruk")          # Tambah di akhir
buah.insert(1, "melon")       # Tambah di posisi ke-1
buah.extend(["nanas", "pepaya"]) # Tambah banyak sekaligus

print(buah)

                    
Output:

                    
['apel', 'melon', 'pisang', 'jeruk', 'nanas', 'pepaya']

                    
ğŸ’¡ append() menambah satu item saja, sedangkan extend()
                        menambah banyak sekaligus.

                


                
                

                    
5ï¸âƒ£ Menghapus Elemen dari List

9ï¸âƒ£ Slicing (Memotong List)

                    
Slicing digunakan untuk mengambil sebagian elemen dari list.

                    
buah = ["apel", "pisang", "jeruk", "mangga", "melon"]
print(buah[1:4])   # ['pisang', 'jeruk', 'mangga']
print(buah[:3])    # ['apel', 'pisang', 'jeruk']
print(buah[2:])    # ['jeruk', 'mangga', 'melon']
print(buah[::-1])  # urutan dibalik

                    
Format umum: list[awal:akhir:langkah]

                


                
                

                    
ğŸ”Ÿ Nested List (List di dalam List)

print(kelas[1][1])  # 90

                    

                        Nested list berguna untuk menyimpan data seperti tabel, nilai siswa, atau data 2 dimensi.
                    

                


                
                

                    
ğŸ” Looping dalam List

                    
Kita bisa menelusuri setiap elemen dalam list menggunakan for.

                    
buah = ["apel", "pisang", "jeruk"]
for item in buah:
    print(item)

                    
Gunakan enumerate() jika ingin tahu posisi dan isinya:

                    
for i, item in enumerate(buah):
    print(i, item)

                


                
                

                    
âš¡ List Comprehension

                    
Cara singkat membuat list dengan pola tertentu.

                    
angka = [x for x in range(5)]
kuadrat = [x**2 for x in range(1,6)]
ganjil = [x for x in range(10) if x % 2 != 0]
print(angka)
print(kuadrat)
print(ganjil)

                    
Output:

                    
[0, 1, 2, 3, 4]
[1, 4, 9, 16, 25]
[1, 3, 5, 7, 9]

                    
ğŸ’¡ Digunakan untuk membuat list baru dari pola atau kondisi tertentu.

                


                
                

                    
ğŸ§© Copy dan Clear List

salinan = data.copy()     # duplikasi list
salinan.append(4)

print("Setelah clear:", data)

                    
Output:

                    
Asli: [1, 2, 3]
Salinan: [1, 2, 3, 4]
Setelah clear: []
