Pengertian List

                
List adalah tipe data Python yang digunakan untuk menyimpan sekumpulan data dalam satu variabel.
                    Elemen di dalam list bisa berupa angka, string, boolean, bahkan list lain.

                
contoh = ["apel", "jeruk", "mangga", 10, True]

                
List bersifat mutable, artinya datanya bisa diubah setelah dibuat.

            


            

                
Membuat dan Mengakses List

                
Gunakan tanda kurung siku [] untuk membuat list.

                
buah = ["apel", "pisang", "mangga"]
print(buah[0])     # apel
print(buah[-1])    # mangga

                
Indeks dimulai dari 0. Nilai negatif digunakan untuk menghitung dari belakang.
                

            


            

                
Menambah Elemen List

                
buah.append("jeruk")      # tambah di akhir
buah.insert(1, "melon")   # tambah di posisi tertentu
buah.extend(["nanas", "pepaya"]) # tambah banyak elemen

                
append() untuk satu data, sedangkan extend() untuk banyak data
                    sekaligus.

            


            

                
Menghapus Elemen List

                
buah.remove("pisang")  # hapus berdasarkan nilai
buah.pop()            # hapus elemen terakhir
del buah[0]           # hapus berdasarkan indeks

                
    
                    
  • remove() â†’ hapus berdasarkan isi.
    • 
                    
  • pop() â†’ hapus & kembalikan elemen terakhir.
    • 
                    
  • del â†’ hapus berdasarkan posisi indeks.
    • 
                

            


            

                
Mengubah Elemen

                
buah[0] = "apel hijau"
buah[1:3] = ["melon", "nanas"]

                
Kamu bisa mengubah satu atau beberapa elemen sekaligus menggunakan slicing.

            


            

                
Operasi Dasar List

                
angka = [1, 2, 3]
print(len(angka))        # 3
print(angka + [4, 5])    # gabung list
print(angka * 2)         # duplikasi
print(2 in angka)        # cek keanggotaan

            


            

                
Fungsi Bawaan List

                
angka = [2, 4, 4, 5, 10]
print(angka.count(4))   # menghitung jumlah 4
print(angka.index(5))   # posisi angka 5
angka.sort()            # urut naik
angka.reverse()         # urut turun
print(angka)

            


            

                
Slicing List

                
buah = ["apel", "pisang", "jeruk", "mangga", "melon"]
print(buah[1:4])   # ['pisang', 'jeruk', 'mangga']
print(buah[:3])    # ['apel', 'pisang', 'jeruk']
print(buah[::-1])  # urutan dibalik

            


            

                
Nested List (List di dalam List)

                
kelas = [["Ani", 80], ["Budi", 90], ["Citra", 85]]
print(kelas[1][0])  # Budi
print(kelas[1][1])  # 90

                
Nested list digunakan untuk menyimpan data tabel atau pasangan nilai.

            


            

                
Looping pada List

                
for buah in ["apel", "pisang", "jeruk"]:
    print(buah)

                
Gunakan enumerate() untuk mendapatkan indeks dan nilai:

                
buah = ["apel", "jeruk", "pisang"]
for i, item in enumerate(buah):
    print(i, item)

            


            

                
List Comprehension

                
angka = [x for x in range(5)]
kuadrat = [x**2 for x in range(1,6)]
print(angka)
print(kuadrat)

            


            

                
Fungsi Copy dan Clear

                
data = [1, 2, 3]
salinan = data.copy()
salinan.append(4)
print("Asli:", data)
print("Salinan:", salinan)

data.clear()
print("Setelah clear:", data)
